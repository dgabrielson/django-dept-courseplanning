Make sure that all teaching faculty have teaching profiles

Home › Programs & Course Info › Teaching profiles
  • Make sure the person associated with the teaching profile has a username – login is required by each teaching faculty member to view or update their teaching profile.

Setup additional teaching survey questions

Home › Programs & Course Info › Teaching survey questions
  • Additional survey questions are meant to provide further information regrading teaching.
  • Example questions are:
    “Are there any other particulars you would like us to consider when assigning your teaching for this upcoming year?”
    “Are there any graduate courses or topics courses you would like to propose?”

Setup programs for scheduling

Home › Programs & Course Info › Programs
  • Programs that are used for scheduling are groups of courses that teaching faculty can rank their preferences for.
  • Each program is one step in the teaching profile, so it's a good idea to have multiple programs for scheduling.
  • All of the programs for scheduling should cover the entire undergraduate program.
    Departments may or may not schedule graduate courses at the same time.

Distribute the teaching survey link

{{ request.scheme }}://{{ request.get_host }}{% url 'courseplanning-teaching-profile' %}
  • Everybody gets the same url, but login is required to identify the profile being updated.

Set up a draft schedule

Home › Programs & Course Info › Draft schedule sessions > Add
  • Start by entering a name and the start and end dates for the draft schedule.
  • At the moment, the scheduler is designed to allow scheduling up to one year with a single draft schedule – specifing a longer time period may have unpredicatible results.
  • Save using the “Save and continue editing” button.
  • Intialize the schedule using one of the availabe options.
  • Depending on the courses available, you may only have one available option for initalizing a draft schedule. This is normal.
  • The “Auto Schedule” option is provided as a convience only – always review using the scheduling worksheet.

Edit the draft sections

Home › Programs & Course Info › Draft sections
  • Using the filters, select the draft schedule and possibly the term you want to focus on.
  • Ensure that there are the desired courses and sections.
  • You will need to manually update draft sections for courses that only run every other year (or irregularly).

Use the scheduling worksheet

Home › Programs & Course Info › Draft schedule sections
  • Using the link above, click the “Worksheet” button.
  • A large screen is helpful when using the worksheet.
  • Clicking the icon beside a course name will show a list of instructor preferences for that course.
  • Clicking an instructor from the right sidebar will show that instructors preferences.
  • As courses are assigned instructors (and timeslots) scores and notes are updated, as is instructor load information in the right sidebar.
  • For each assignment, higher scores represent more desireable options.
